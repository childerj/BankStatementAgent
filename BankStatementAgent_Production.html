🏢 Overview

The BankStatementAgent is an autonomous AI-powered system deployed on Azure that automatically processes bank statement PDFs and converts them to BAI2 format for banking system integration. The system is now production-ready with enhanced scalability for processing hundreds of files from hundreds of banks simultaneously.

Key Features: Autonomous Processing • AI-Powered Extraction • OpenAI Integration • Format Conversion • Error Resilience • Complete Audit Trail • Real-time Monitoring • Multi-Bank Scalability

🚀 Deployment Guide

Prerequisites

  • Azure Subscription with Function Apps enabled
  • Azure CLI installed and configured
  • Python 3.10+ development environment
  • OpenAI API access for routing enhancement

Production Deployment Commands

# Login to Azure az login # Create resource group az group create --name rg-bankstatement --location "East US" # Create Function App (Linux Consumption for EventGrid) az functionapp create \ --resource-group rg-bankstatement \ --consumption-plan-location "East US" \ --runtime python \ --runtime-version 3.10 \ --functions-version 4 \ --name BankStatementAgent \ --storage-account waazuse1aistorage \ --os-type Linux # Deploy function with enhancements func azure functionapp publish BankStatementAgent # Configure EventGrid subscription az eventgrid event-subscription create \ --name bank-statement-processor \ --source-resource-id "/subscriptions/{sub-id}/resourceGroups/rg-bankstatement/providers/Microsoft.Storage/storageAccounts/waazuse1aistorage" \ --endpoint-type azurefunction \ --endpoint "/subscriptions/{sub-id}/resourceGroups/rg-bankstatement/providers/Microsoft.Web/sites/BankStatementAgent/functions/process_new_file"
